# Leetcode
### 根据数据结构和算法进行分类

### 代码书写注意：
运算符前后用空格；冒号、逗号后用空格；括号内侧不加空格；
函数和类之间两行空行；同一函数内的逻辑块之间用一个空行分隔；
类名首字母大写；常量全大写+下划线；变量、函数、方法小写字母+下划线；
单行长度不超过 88 个字符；
使用显式布尔判断：避免 if len(list) > 0，直接用 if list；

### 文件命名一般用-（中划线）
驼峰命名法：大驼峰UpperCamelCase（类）；小驼峰lowerCamelCase（变量，函数）；蛇形命名法sorted_squares.py（文件）；

## 数组 Array

- 数组是存放在连续内存空间上的相同类型数据的集合。

- 那么二维数组在内存的空间地址是连续的么？不同语言的设计不同，自己搜搜看。

>
